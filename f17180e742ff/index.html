Go иҜӯиЁҖеҺҹеӯҗж“ҚдҪң (Atomic Operations) жҸҗдҫӣдәҶдёҖз§ҚеңЁе№¶еҸ‘зҺҜеўғдёӯеҜ№е…ұдә«еҸҳйҮҸиҝӣиЎҢе®үе…ЁгҖҒй«ҳж•Ҳи®ҝй—®зҡ„жңәеҲ¶гҖӮдёҺдј з»ҹзҡ„дә’ж–Ҙй”Ғ (Mutex) дёҚеҗҢпјҢеҺҹеӯҗж“ҚдҪңжҳҜж— й”Ғ (lock-free) зҡ„гҖӮе®ғд»¬йҖҡиҝҮзЎ¬д»¶жҢҮд»ӨдҝқиҜҒж“ҚдҪңзҡ„еҺҹеӯҗжҖ§пјҢеҚідёҖдёӘж“ҚдҪңеңЁжү§иЎҢиҝҮзЁӢдёӯдёҚдјҡиў«е…¶д»–е№¶еҸ‘ж“ҚдҪңжү“ж–ӯгҖӮиҝҷдҪҝеҫ—еҺҹеӯҗж“ҚдҪңеңЁжҹҗдәӣеңәжҷҜдёӢжҜ”дә’ж–Ҙй”Ғе…·жңүжӣҙй«ҳзҡ„жҖ§иғҪпјҢеӣ дёәе®ғд»¬йҒҝе…ҚдәҶж“ҚдҪңзі»з»ҹдёҠдёӢж–ҮеҲҮжҚўе’Ңй”Ғз«һдәүеёҰжқҘзҡ„ејҖй”ҖгҖӮеҺҹеӯҗж“ҚдҪңдё»иҰҒз”ЁдәҺжӣҙж–°еҹәжң¬ж•°жҚ®зұ»еһӢпјҲеҰӮж•ҙж•°гҖҒжҢҮй’Ҳпјүзҡ„е…ұдә«еҖјпјҢд»ҘйҒҝе…Қз«һжҖҒжқЎд»¶ (race condition)гҖӮ

ж ёеҝғжҖқжғіпјҡ

  • ж— й”Ғе№¶еҸ‘пјҡдёҚдҪҝз”Ёдә’ж–Ҙй”ҒпјҢзӣҙжҺҘеҲ©з”Ё CPU жҢҮд»ӨдҝқиҜҒж“ҚдҪңе®Ңж•ҙжҖ§гҖӮ
  • еҺҹеӯҗжҖ§пјҡж“ҚдҪңиҰҒд№Ҳе®Ңе…ЁжҲҗеҠҹпјҢиҰҒд№Ҳж №жң¬дёҚеҸ‘з”ҹпјҢдёӯй—ҙзҠ¶жҖҒеҜ№е…¶д»–зәҝзЁӢдёҚеҸҜи§ҒгҖӮ
  • ж•ҲзҺҮй«ҳпјҡйҒҝе…ҚдәҶй”Ғзҡ„ејҖй”ҖпјҲеҰӮдёҠдёӢж–ҮеҲҮжҚўпјүпјҢеңЁдҪҺз«һдәүеңәжҷҜдёӢиЎЁзҺ°еҮәиүІгҖӮ
  • жӣҝд»Јдә’ж–Ҙй”ҒпјҡеҪ“е…ұдә«ж•°жҚ®жҳҜеҚ•дёӘеҹәжң¬зұ»еһӢж—¶пјҢеҺҹеӯҗж“ҚдҪңжҳҜдә’ж–Ҙй”Ғзҡ„иҪ»йҮҸзә§жӣҝд»Јж–№жЎҲгҖӮ

дёҖгҖҒдёәд»Җд№ҲйңҖиҰҒеҺҹеӯҗж“ҚдҪңпјҹе№¶еҸ‘зј–зЁӢй—®йўҳ

еңЁ Go иҜӯиЁҖдёӯпјҢGoroutine жҳҜиҪ»йҮҸзә§зҡ„е№¶еҸ‘жү§иЎҢеҚ•е…ғгҖӮеҪ“еӨҡдёӘ Goroutine еҗҢж—¶и®ҝй—®е’Ңдҝ®ж”№еҗҢдёҖдёӘе…ұдә«еҸҳйҮҸж—¶пјҢеҰӮжһңжІЎжңүйҖӮеҪ“зҡ„еҗҢжӯҘжңәеҲ¶пјҢе°ұдјҡеҜјиҮҙз«һжҖҒжқЎд»¶ (Race Condition)гҖӮз«һжҖҒжқЎд»¶дјҡеҜјиҮҙзЁӢеәҸзҡ„иЎҢдёәдёҚеҸҜйў„жөӢпјҢдә§з”ҹй”ҷиҜҜзҡ„з»“жһңгҖӮ

зӨәдҫӢпјҡз«һжҖҒжқЎд»¶

еңЁдёҠйқўзҡ„дҫӢеӯҗдёӯпјҢcounter++ е№¶дёҚжҳҜдёҖдёӘеҺҹеӯҗж“ҚдҪңгҖӮе®ғе®һйҷ…дёҠеҢ…еҗ«дёүдёӘжӯҘйӘӨпјҡ

  1. иҜ»еҸ– counter зҡ„еҪ“еүҚеҖјгҖӮ
  2. е°ҶиҜ»еҸ–зҡ„еҖјеҠ  1гҖӮ
  3. е°Ҷж–°еҖјеҶҷеӣһ counterгҖӮ

еҪ“еӨҡдёӘ Goroutine еҗҢж—¶жү§иЎҢиҝҷдёүдёӘжӯҘйӘӨж—¶пјҢеҸҜиғҪдјҡеҸ‘з”ҹд»ҘдёӢжғ…еҶөпјҡ

  • Goroutine A иҜ»еҸ– counter = 0гҖӮ
  • Goroutine B иҜ»еҸ– counter = 0гҖӮ
  • Goroutine A е°Ҷ counter жӣҙж–°дёә 1гҖӮ
  • Goroutine B е°Ҷ counter жӣҙж–°дёә 1пјҲиҖҢдёҚжҳҜ 2пјүгҖӮ
    жңҖз»ҲеҜјиҮҙ counter зҡ„еҖјжҜ”йў„жңҹзҡ„е°ҸгҖӮ

дёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёдә’ж–Ҙй”Ғ sync.Mutex жқҘдҝқжҠӨ counter зҡ„и®ҝй—®гҖӮ

зӨәдҫӢпјҡдҪҝз”Ё Mutex и§ЈеҶіз«һжҖҒжқЎд»¶

дҪҝз”Ёдә’ж–Ҙй”ҒеҸҜд»ҘзЎ®дҝқе№¶еҸ‘е®үе…ЁпјҢдҪҶй”Ғзҡ„еҠ й”Ғе’Ңи§Јй”Ғж“ҚдҪңдјҡеёҰжқҘдёҖе®ҡзҡ„жҖ§иғҪејҖй”ҖпјҲдҫӢеҰӮпјҢдёҠдёӢж–ҮеҲҮжҚўгҖҒзј“еӯҳеҗҢжӯҘпјүгҖӮеңЁжҹҗдәӣз®ҖеҚ•ж“ҚдҪңпјҲеҰӮйҖ’еўһ/йҖ’еҮҸгҖҒеҠ иҪҪ/еӯҳеӮЁпјүдёҠпјҢеҺҹеӯҗж“ҚдҪңжҸҗдҫӣдәҶжӣҙиҪ»йҮҸзә§зҡ„жӣҝд»Јж–№жЎҲгҖӮ

дәҢгҖҒGo иҜӯиЁҖ sync/atomic еҢ…

Go иҜӯиЁҖйҖҡиҝҮ sync/atomic еҢ…жҸҗдҫӣдәҶдёҖзі»еҲ—еҺҹеӯҗж“ҚдҪңеҮҪж•°пјҢиҝҷдәӣеҮҪж•°зӣҙжҺҘжҳ е°„еҲ° CPU дҪ“зі»з»“жһ„жҸҗдҫӣзҡ„еҺҹеӯҗжҢҮд»ӨпјҢд»ҺиҖҢе®һзҺ°ж— й”Ғзҡ„е№¶еҸ‘и®ҝй—®гҖӮ

sync/atomic еҢ…дё»иҰҒжҸҗдҫӣдәҶд»ҘдёӢеҮ зұ»еҺҹеӯҗж“ҚдҪңпјҡ

  1. еўһ/еҮҸж“ҚдҪң (Add)пјҡеҺҹеӯҗең°еўһеҠ жҲ–еҮҸе°‘дёҖдёӘж•ҙж•°еҖјгҖӮ
  2. жҜ”иҫғе№¶дәӨжҚў (Compare And Swap, CAS)пјҡеҺҹеӯҗең°жҜ”иҫғдёҖдёӘеҖјжҳҜеҗҰзӯүдәҺйў„жңҹеҖјпјҢеҰӮжһңзӣёзӯүеҲҷжӣҙж–°дёәж–°еҖјгҖӮ
  3. еҠ иҪҪ (Load)пјҡеҺҹеӯҗең°иҜ»еҸ–дёҖдёӘеҖјгҖӮ
  4. еӯҳеӮЁ (Store)пјҡеҺҹеӯҗең°еҶҷе…ҘдёҖдёӘеҖјгҖӮ
  5. дәӨжҚў (Swap)пјҡеҺҹеӯҗең°е°ҶдёҖдёӘеҖјжӣҝжҚўдёәж–°еҖјпјҢе№¶иҝ”еӣһж—§еҖјгҖӮ

ж”ҜжҢҒзҡ„зұ»еһӢеҢ…жӢ¬ int32, int64, uint32, uint64, uintptr (з”ЁдәҺжҢҮй’Ҳж“ҚдҪң) е’Ң atomic.Pointer (Go 1.19+пјҢз”ЁдәҺжіӣеһӢжҢҮй’Ҳж“ҚдҪң)гҖӮ

2.1 еўһ/еҮҸж“ҚдҪң (Add)

AddInt32, AddInt64, AddUint32, AddUint64, AddUintptr зӯүеҮҪж•°з”ЁдәҺеҺҹеӯҗең°еўһеҠ жҲ–еҮҸе°‘жҢҮе®ҡең°еқҖзҡ„еҖјгҖӮ

зӨәдҫӢпјҡдҪҝз”Ё atomic.AddInt64 и§ЈеҶіз«һжҖҒжқЎд»¶

зӨәдҫӢпјҡдҪҝз”Ё CAS е®һзҺ°иҮӘж—Ӣй”ҒжҲ–ж— й”Ғж Ҳ

CAS еёёз”ЁдәҺе®һзҺ°д№җи§Ӯй”Ғзҡ„ж•°жҚ®з»“жһ„пјҢжҜ”еҰӮж— й”ҒйҳҹеҲ—жҲ–ж— й”Ғж ҲгҖӮ

2.3 еҠ иҪҪ (Load)

LoadInt32, LoadInt64, LoadUint32, LoadUint64, LoadUintptr, LoadPointerгҖӮ

еҺҹеӯҗең°иҜ»еҸ–жҢҮе®ҡең°еқҖзҡ„еҖјгҖӮиҝҷдёҚд»…жҳҜиҜ»еҸ–пјҢжӣҙйҮҚиҰҒзҡ„жҳҜе®ғжҸҗдҫӣдәҶеҶ…еӯҳеұҸйҡң (Memory Barrier)пјҢзЎ®дҝқдәҶеңЁ Load д№ӢеүҚзҡ„жүҖжңүеҶҷе…Ҙж“ҚдҪңеҜ№еҪ“еүҚ Goroutine йғҪжҳҜеҸҜи§Ғзҡ„пјҢйҒҝе…ҚдәҶзј–иҜ‘еҷЁе’Ң CPU зҡ„йҮҚжҺ’еәҸдјҳеҢ–й—®йўҳгҖӮ

зӨәдҫӢпјҡеҺҹеӯҗиҜ»еҸ–жңҖж–°еҖј

2.4 еӯҳеӮЁ (Store)

StoreInt32, StoreInt64, StoreUint32, StoreUint64, StoreUintptr, StorePointerгҖӮ

еҺҹеӯҗең°еҶҷе…ҘжҢҮе®ҡең°еқҖзҡ„еҖјгҖӮдёҺ Load зұ»дјјпјҢе®ғд№ҹжҸҗдҫӣдәҶеҶ…еӯҳеұҸйҡңпјҢзЎ®дҝқеңЁ Store д№ӢеҗҺпјҢд№ӢеүҚзҡ„еҶҷе…Ҙж“ҚдҪңеҜ№д»»дҪ•е…¶д»– Goroutine йғҪеҸҜи§ҒгҖӮ

зӨәдҫӢи§Ғ Load зӨәдҫӢгҖӮ

2.5 дәӨжҚў (Swap)

SwapInt32, SwapInt64, SwapUint32, SwapUint64, SwapUintptr, SwapPointerгҖӮ

еҺҹеӯҗең°е°Ҷ addr жҢҮеҗ‘зҡ„еҖјжӣҝжҚўдёә new еҖјпјҢе№¶иҝ”еӣһ addr зҡ„ж—§еҖјгҖӮ

зӨәдҫӢпјҡеҺҹеӯҗдәӨжҚўеҖј

atomic.Pointer[T] (Go 1.19+)

Go 1.19 еј•е…ҘдәҶжіӣеһӢзҡ„ atomic.Pointer[T] зұ»еһӢпјҢдҪҝеҫ—еҜ№д»»ж„Ҹзұ»еһӢжҢҮй’Ҳзҡ„еҺҹеӯҗж“ҚдҪңжӣҙдёәж–№дҫҝе’Ңзұ»еһӢе®үе…ЁгҖӮе®ғе°ҒиЈ…дәҶеҜ№ uintptr зҡ„ Load, Store, Swap, CompareAndSwap ж“ҚдҪңгҖӮ

дёүгҖҒеҺҹеӯҗж“ҚдҪңдёҺдә’ж–Ҙй”Ғзҡ„йҖүжӢ©

зү№жҖ§ еҺҹеӯҗж“ҚдҪң (sync/atomic) дә’ж–Ҙй”Ғ (sync.Mutex)
зІ’еәҰ зІҫз»ҶпјҢйҖҡеёёй’ҲеҜ№еҚ•дёӘеҹәжң¬зұ»еһӢеҸҳйҮҸ зІ—зіҷпјҢеҸҜд»ҘдҝқжҠӨдёҖж®өд»Јз Ғеқ—пјҢеҢ…еҗ«еӨҡдёӘеҸҳйҮҸе’ҢеӨҚжқӮйҖ»иҫ‘
жҖ§иғҪ й«ҳпјҢж— й”ҒпјҢзӣҙжҺҘеҲ©з”Ё CPU жҢҮд»ӨпјҢж— дёҠдёӢж–ҮеҲҮжҚўејҖй”Җ дҪҺдәҺеҺҹеӯҗж“ҚдҪңпјҢж¶үеҸҠ Goroutine и°ғеәҰгҖҒдёҠдёӢж–ҮеҲҮжҚўзӯүејҖй”Җ
йҖӮз”ЁеңәжҷҜ еҜ№еҚ•дёӘз®ҖеҚ•е…ұдә«еҸҳйҮҸпјҲеҰӮи®Ўж•°еҷЁгҖҒзҠ¶жҖҒж Үеҝ—пјүзҡ„з®ҖеҚ•ж“ҚдҪң дҝқжҠӨеӨҚжқӮзҡ„ж•°жҚ®з»“жһ„гҖҒеӨҡеҸҳйҮҸзҠ¶жҖҒгҖҒйңҖиҰҒй•ҝж—¶й—ҙдҝқжҢҒзҡ„дёҙз•ҢеҢә
еӨҚжқӮжҖ§ жҳ“дәҺдҪҝз”ЁпјҢдҪҶе®һзҺ°еӨҚжқӮйҖ»иҫ‘йңҖиҰҒз»„еҗҲеӨҡдёӘ CAS ж“ҚдҪң з®ҖеҚ•пјҢд»Ҙ Lock() е’Ң Unlock() дҝқжҠӨд»Јз Ғеқ—
жӯ»й”Ғ дёҚдјҡдә§з”ҹжӯ»й”Ғ еҸҜиғҪдә§з”ҹжӯ»й”ҒпјҢйңҖиҰҒе°Ҹеҝғз®ЎзҗҶ
еҶ…еӯҳеұҸйҡң жҸҗдҫӣеҶ…еӯҳеұҸйҡңпјҢзЎ®дҝқеҸҜи§ҒжҖ§е’ҢжңүеәҸжҖ§ йҡҗејҸжҸҗдҫӣеҶ…еӯҳеұҸйҡң

д»Җд№Ҳж—¶еҖҷдҪҝз”ЁеҺҹеӯҗж“ҚдҪңпјҹ

  • еҪ“йңҖиҰҒжӣҙж–°зҡ„е…ұдә«ж•°жҚ®жҳҜеҚ•дёӘеҹәжң¬зұ»еһӢпјҲеҰӮ int32, int64, жҢҮй’ҲпјүгҖӮ
  • ж“ҚдҪңжҳҜз®ҖеҚ•зҡ„еўһ/еҮҸгҖҒеҠ иҪҪгҖҒеӯҳеӮЁгҖҒдәӨжҚўжҲ–жҜ”иҫғе№¶дәӨжҚўгҖӮ
  • иҝҪжұӮжһҒиҮҙжҖ§иғҪпјҢе№¶дё”й”Ғз«һдәүжҝҖзғҲеәҰиҫғдҪҺзҡ„еңәжҷҜгҖӮ
  • е®һзҺ°ж— й”Ғж•°жҚ®з»“жһ„ (lock-free data structures)гҖӮ

д»Җд№Ҳж—¶еҖҷдҪҝз”Ёдә’ж–Ҙй”Ғпјҹ

  • йңҖиҰҒдҝқжҠӨеӨҡдёӘе…ұдә«еҸҳйҮҸзҡ„ж“ҚдҪңпјҢжҲ–иҖ…дёҙз•ҢеҢәеҢ…еҗ«еӨҡиЎҢд»Јз Ғе’ҢеӨҚжқӮйҖ»иҫ‘ж—¶гҖӮ
  • йңҖиҰҒдҝқжҠӨйқһеҹәжң¬зұ»еһӢзҡ„ж•°жҚ®з»“жһ„пјҲеҰӮ map, slice, ``struct`пјүгҖӮ
  • ж“ҚдҪңиҖ—ж—¶иҫғй•ҝпјҢCPU еҜҶйӣҶеһӢжҲ– I/O еҜҶйӣҶеһӢж“ҚдҪңгҖӮ
  • еҲқеӯҰиҖ…пјҢдә’ж–Ҙй”Ғзҡ„зҗҶи§Је’ҢдҪҝз”ЁзӣёеҜ№зӣҙи§ӮпјҢдёҚе®№жҳ“еҮәй”ҷгҖӮ
