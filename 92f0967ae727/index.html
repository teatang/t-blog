PHP йҮҚеҶҷ Trait ж–№жі•иҜҰи§Ј

еҸ‘иЎЁдәҺ2023-10-21|PHP
|жҖ»еӯ—ж•°:2.7k|йҳ…иҜ»ж—¶й•ҝ:10еҲҶй’ҹ

PHP дёӯзҡ„ Trait (зү№иҙЁ) жҳҜдёҖз§Қд»Јз ҒеӨҚз”ЁжңәеҲ¶пјҢе®ғе…Ғи®ёдҪ е°ҶдёҖз»„ж–№жі•жҸ’е…ҘеҲ°еӨҡдёӘдёҚзӣёе…ізҡ„зұ»дёӯпјҢд»ҺиҖҢи§ЈеҶіеҚ•з»§жүҝиҜӯиЁҖдёӯд»Јз Ғе…ұдә«зҡ„йҷҗеҲ¶гҖӮTrait еј•е…ҘдәҶдёҖз§Қж°ҙе№іеӨҚз”Ё (Horizontal Reuse) зҡ„ж–№ејҸпјҢдёҺдј з»ҹзҡ„еһӮзӣҙз»§жүҝ (Vertical Inheritance) еҪўжҲҗдә’иЎҘгҖӮеҪ“дёҖдёӘзұ» use дәҶдёҖдёӘ Trait еҗҺпјҢTrait дёӯзҡ„ж–№жі•е°ұеҰӮеҗҢеңЁзұ»дёӯеЈ°жҳҺдёҖж ·гҖӮз„¶иҖҢпјҢеңЁжҹҗдәӣжғ…еҶөдёӢпјҢжҲ‘д»¬еҸҜиғҪйңҖиҰҒеҜ№ Trait дёӯеј•е…Ҙзҡ„ж–№жі•иҝӣиЎҢйҮҚеҶҷжҲ–и°ғж•ҙгҖӮжң¬ж–Үе°ҶиҜҰз»ҶжҺўи®Ё PHP дёӯеҰӮдҪ•йҮҚеҶҷ Trait ж–№жі•зҡ„еҗ„з§Қзӯ–з•Ҙе’Ңдјҳе…Ҳзә§и§„еҲҷгҖӮ

ж ёеҝғжҰӮеҝөпјҡ

  • TraitпјҡдёҖз»„еҸҜеӨҚз”Ёзҡ„ж–№жі•йӣҶеҗҲпјҢйҖҡиҝҮ use е…ій”®еӯ—ж··е…Ҙзұ»дёӯгҖӮ
  • ж–№жі•йҮҚеҶҷдјҳе…Ҳзә§пјҡзұ»иҮӘиә«ж–№жі• > Trait ж–№жі• > зҲ¶зұ»ж–№жі•гҖӮ
  • еҶІзӘҒи§ЈеҶіпјҡinsteadof е’Ң as е…ій”®еӯ—з”ЁдәҺеӨ„зҗҶеӨҡдёӘ Trait д№Ӣй—ҙжҲ– Trait дёҺзұ»ж–№жі•д№Ӣй—ҙзҡ„еҗҚз§°еҶІзӘҒгҖӮ

дёҖгҖҒTrait зҡ„еҹәжң¬жҰӮеҝөеӣһйЎҫ

Trait ж—ЁеңЁеҮҸе°‘еҚ•з»§жүҝиҜӯиЁҖзҡ„йҷҗеҲ¶пјҢе®ғе…Ғи®ёејҖеҸ‘иҖ…иҮӘз”ұең°з»„еҗҲеҠҹиғҪпјҢиҖҢж— йңҖйҖҡиҝҮеӨҚжқӮзҡ„з»§жүҝеұӮж¬Ўз»“жһ„гҖӮ

зӨәдҫӢпјҡ

дәҢгҖҒTrait ж–№жі•зҡ„дјҳе…Ҳзә§и§„еҲҷ

еҪ“дёҖдёӘзұ» use дәҶ TraitпјҢе№¶дё”зұ»дёӯгҖҒTrait дёӯжҲ–зҲ¶зұ»дёӯеӯҳеңЁеҗҢеҗҚж–№жі•ж—¶пјҢPHP йҒөеҫӘдёҘж јзҡ„дјҳе…Ҳзә§и§„еҲҷжқҘеҶіе®ҡе“ӘдёӘж–№жі•дјҡиў«е®һйҷ…дҪҝз”Ёпјҡ

  1. еҪ“еүҚзұ»дёӯзҡ„ж–№жі• > Trait дёӯзҡ„ж–№жі•
    еҰӮжһңзұ»иҮӘиә«е®ҡд№үдәҶдёҖдёӘдёҺ Trait дёӯеҗҢеҗҚзҡ„ж–№жі•пјҢйӮЈд№Ҳзұ»дёӯе®ҡд№үзҡ„ж–№жі•дјҡиҰҶзӣ– Trait дёӯзҡ„ж–№жі•гҖӮ

  2. Trait дёӯзҡ„ж–№жі• > зҲ¶зұ»дёӯзҡ„ж–№жі•
    еҰӮжһңдёҖдёӘзұ»з»§жүҝдәҶдёҖдёӘзҲ¶зұ»пјҢе№¶дё” Trait дёӯзҡ„ж–№жі•дёҺзҲ¶зұ»дёӯзҡ„ж–№жі•еҗҢеҗҚпјҢйӮЈд№Ҳ Trait дёӯзҡ„ж–№жі•дјҡиҰҶзӣ–зҲ¶зұ»дёӯзҡ„ж–№жі•гҖӮ

дјҳе…Ҳзә§йЎәеәҸжҖ»з»“пјҡ еҪ“еүҚзұ»ж–№жі• > Trait ж–№жі• > зҲ¶зұ»ж–№жі•

2.1 зӨәдҫӢпјҡзұ»ж–№жі•иҰҶзӣ– Trait ж–№жі•

2.2 зӨәдҫӢпјҡTrait ж–№жі•иҰҶзӣ–зҲ¶зұ»ж–№жі•

дёүгҖҒи§ЈеҶі Trait ж–№жі•еҶІзӘҒ

еҪ“дёҖдёӘзұ» use еӨҡдёӘ Trait ж—¶пјҢеҰӮжһңиҝҷдәӣ Trait дёӯеӯҳеңЁеҗҢеҗҚж–№жі•пјҢе°ұдјҡеҸ‘з”ҹеҶІзӘҒгҖӮPHP жҸҗдҫӣдәҶ insteadof е’Ң as е…ій”®еӯ—жқҘи§ЈеҶіиҝҷдәӣеҶІзӘҒгҖӮ

3.1 insteadofпјҡжҳҺзЎ®йҖүжӢ©дёҖдёӘ Trait ж–№жі•

insteadof е…ій”®еӯ—з”ЁдәҺжҢҮе®ҡеңЁеҸ‘з”ҹж–№жі•еҶІзӘҒж—¶пјҢйҖүжӢ©е“ӘдёӘ Trait дёӯзҡ„ж–№жі•жқҘдҪҝз”ЁпјҢе№¶еҝҪз•Ҙе…¶д»– Trait дёӯзҡ„еҗҢеҗҚж–№жі•гҖӮ

зӨәдҫӢпјҡ

иҜҙжҳҺпјҡTraitA::doSomething insteadof TraitB; иҝҷиЎҢд»Јз Ғе‘ҠиҜү PHP иҝҗиЎҢж—¶пјҢеҪ“ MyConflictClass дёӯи°ғз”Ё doSomething ж–№жі•ж—¶пјҢдҪҝз”Ё TraitA жҸҗдҫӣзҡ„зүҲжң¬пјҢиҖҢеҝҪз•Ҙ TraitB зҡ„зүҲжң¬гҖӮ

3.2 asпјҡдёәеҶІзӘҒж–№жі•еҲӣе»әеҲ«еҗҚжҲ–дҝ®ж”№еҸҜи§ҒжҖ§

as е…ій”®еӯ—жңүдёӨдёӘдё»иҰҒз”ЁйҖ”пјҡ

  1. дёәеҶІзӘҒж–№жі•еҲӣе»әеҲ«еҗҚпјҡеҪ“дҪ еёҢжңӣеҗҢж—¶дҪҝз”ЁдёӨдёӘеҶІзӘҒзҡ„ Trait ж–№жі•ж—¶пјҢеҸҜд»Ҙдёәе…¶дёӯдёҖдёӘжҲ–дёӨдёӘеҲӣе»әеҲ«еҗҚгҖӮ
  2. дҝ®ж”№ Trait ж–№жі•зҡ„еҸҜи§ҒжҖ§пјҡеҸҜд»Ҙж”№еҸҳж··е…Ҙзұ»дёӯ Trait ж–№жі•зҡ„и®ҝй—®дҝ®йҘ°з¬Ұ (public, protected, private)гҖӮ

зӨәдҫӢ 1пјҡдёәеҶІзӘҒж–№жі•еҲӣе»әеҲ«еҗҚ

иҜҙжҳҺпјҡиҝҷйҮҢжҲ‘д»¬йҰ–е…ҲдҪҝз”Ё insteadof и§ЈеҶідәҶ doSomething зҡ„еҶІзӘҒпјҢйҖүжӢ©дәҶ TraitA зҡ„зүҲжң¬гҖӮз„¶еҗҺпјҢйҖҡиҝҮ TraitB::doSomething as doSomethingFromB; дёә TraitB зҡ„ doSomething ж–№жі•еҲӣе»әдәҶдёҖдёӘж–°зҡ„еҗҚз§° doSomethingFromBпјҢдҪҝеҫ—жҲ‘д»¬еҸҜд»ҘеңЁ MyAliasClass дёӯеҗҢж—¶и®ҝй—®иҝҷдёӨдёӘж–№жі•гҖӮ

зӨәдҫӢ 2пјҡдҝ®ж”№ Trait ж–№жі•зҡ„еҸҜи§ҒжҖ§

иҜҙжҳҺпјҡas е…ій”®еӯ—еҸҜд»ҘеңЁйҮҚе‘ҪеҗҚзҡ„еҗҢж—¶дҝ®ж”№еҸҜи§ҒжҖ§пјҢжҲ–иҖ…еҸӘдҝ®ж”№еҸҜи§ҒжҖ§иҖҢдёҚйҮҚе‘ҪеҗҚгҖӮ

  • utilityMethod as private; е°Ҷ utilityMethod зҡ„и®ҝй—®жқғйҷҗд»Һ public дҝ®ж”№дёә privateгҖӮ
  • internalMethod as public changedInternalMethod; е°Ҷ internalMethod зҡ„и®ҝй—®жқғйҷҗд»Һ protected дҝ®ж”№дёә publicпјҢе№¶е°Ҷе…¶йҮҚе‘ҪеҗҚдёә changedInternalMethodгҖӮ

еӣӣгҖҒеҰӮдҪ•еңЁзұ»дёӯвҖңйҮҚеҶҷвҖқ Trait ж–№жі•

вҖңйҮҚеҶҷвҖқ Trait ж–№жі•еңЁ PHP дёӯе®һйҷ…дёҠжҳҜеҲ©з”ЁдәҶдёҠйқўжҸҗеҲ°зҡ„дјҳе…Ҳзә§и§„еҲҷпјҡзұ»иҮӘиә«е®ҡд№үзҡ„ж–№жі•дјҡиҰҶзӣ– Trait жҸҗдҫӣзҡ„еҗҢеҗҚж–№жі•гҖӮ

еҰӮжһңдҪ жғіеңЁзұ»дёӯдҪҝз”Ё Trait жҸҗдҫӣзҡ„ж–№жі•пјҢдҪҶеҸҲжғіеңЁе…¶еҹәзЎҖдёҠеўһеҠ дёҖдәӣйҖ»иҫ‘жҲ–еҪ»еә•жӣҝжҚўе®ғпјҢдҪ еҸҜд»ҘеңЁзұ»дёӯйҮҚж–°е®ҡд№үиҜҘж–№жі•гҖӮ

4.1 е®Ңж•ҙжӣҝжҚў Trait ж–№жі•

иҝҷжҳҜжңҖзӣҙжҺҘзҡ„вҖңйҮҚеҶҷвҖқж–№ејҸпјҡеңЁзұ»дёӯеЈ°жҳҺдёҖдёӘдёҺ Trait ж–№жі•еҗҢеҗҚзҡ„ж–№жі•гҖӮ

4.2 жү©еұ• Trait ж–№жі• (дҪҝз”Ё parent жҲ– self жЁЎжӢҹ)

Trait дёҚж”ҜжҢҒеғҸз»§жүҝйӮЈж ·зӣҙжҺҘдҪҝз”Ё parent::method() жқҘи°ғз”Ёиў«иҰҶзӣ–зҡ„ж–№жі•гҖӮдҪҶжҳҜпјҢеҰӮжһңдҪ еёҢжңӣеңЁзұ»зҡ„ж–№жі•дёӯи°ғз”Ёиў«иҰҶзӣ–зҡ„ Trait ж–№жі•пјҢдҪ йңҖиҰҒдҪҝз”Ё as е…ій”®еӯ—дёә Trait ж–№жі•еҲӣе»әдёҖдёӘеҲ«еҗҚпјҢз„¶еҗҺеңЁзұ»зҡ„ж–№жі•дёӯи°ғз”ЁиҝҷдёӘеҲ«еҗҚгҖӮ

иҝҷжҳҜжңҖеёёи§Ғзҡ„вҖңйҮҚеҶҷе№¶жү©еұ•вҖқ Trait ж–№жі•зҡ„еңәжҷҜгҖӮ

иҜҙжҳҺпјҡ

  1. жҲ‘д»¬йҰ–е…ҲйҖҡиҝҮ LoggingTrait::logMessage as private logMessageTrait; е°Ҷ Trait дёӯзҡ„ logMessage ж–№жі•еј•е…Ҙзұ»дёӯпјҢе№¶дёәе…¶еҲӣе»әдёҖдёӘ private зҡ„еҲ«еҗҚ logMessageTraitгҖӮиҝҷйҮҢдҪҝз”Ё private жҳҜдёҖдёӘеҘҪд№ жғҜпјҢиЎЁзӨәиҝҷдёӘеҲ«еҗҚеҸӘдҫӣзұ»еҶ…йғЁзҡ„вҖңйҮҚеҶҷвҖқж–№жі•дҪҝз”ЁпјҢдёҚеёҢжңӣеӨ–йғЁзӣҙжҺҘи°ғз”ЁгҖӮ
  2. з„¶еҗҺеңЁ MyService зұ»дёӯе®ҡд№үдәҶдёҖдёӘеҗҢеҗҚзҡ„ public function logMessage($message)гҖӮ
  3. еңЁиҝҷдёӘвҖңйҮҚеҶҷвҖқж–№жі•еҶ…йғЁпјҢжҲ‘д»¬йҖҡиҝҮ $this->logMessageTrait(...) и°ғз”ЁдәҶ Trait жҸҗдҫӣзҡ„еҺҹе§ӢеҠҹиғҪпјҢе№¶еңЁе…¶еүҚеҗҺж·»еҠ дәҶжҲ‘д»¬иҮӘе®ҡд№үзҡ„йҖ»иҫ‘гҖӮ

иҝҷз§ҚжЁЎејҸе…Ғи®ёдҪ еңЁдҝқжҢҒ Trait ж ёеҝғеҠҹиғҪзҡ„еҗҢж—¶пјҢзҒөжҙ»ең°еңЁдёҚеҗҢзҡ„зұ»дёӯеҜ№е…¶иҝӣиЎҢе®ҡеҲ¶еҢ–жү©еұ•пјҢиҝҷеңЁе®һйҷ…ејҖеҸ‘дёӯйқһеёёжңүз”ЁгҖӮ

дә”гҖҒжіЁж„ҸдәӢйЎ№

  • еҸҜи§ҒжҖ§пјҡйҖҡиҝҮ as е…ій”®еӯ—дҝ®ж”№ Trait ж–№жі•зҡ„еҸҜи§ҒжҖ§ж—¶пјҢеҸӘиғҪе°Ҷе…¶д»Һжӣҙе®Ҫжқҫзҡ„и®ҝй—®жқғйҷҗжӣҙж”№дёәжӣҙдёҘж јзҡ„и®ҝй—®жқғйҷҗ (дҫӢеҰӮ public -> protected -> private)пјҢдёҚиғҪеҸҚиҝҮжқҘгҖӮдҫӢеӨ–жҳҜпјҢеҰӮжһңдҪ е°ҶдёҖдёӘ protected ж–№жі•йҮҚе‘ҪеҗҚдёә publicпјҢиҝҷжҳҜе…Ғи®ёзҡ„пјҢеҰӮ internalMethod as public changedInternalMethod; жүҖзӨәгҖӮ
  • жҠҪиұЎж–№жі•пјҡTrait еҸҜд»ҘеҢ…еҗ«жҠҪиұЎж–№жі•гҖӮеҪ“дёҖдёӘзұ» use еҢ…еҗ«жҠҪиұЎж–№жі•зҡ„ Trait ж—¶пјҢиҜҘзұ»еҝ…йЎ»е®һзҺ°иҝҷдәӣжҠҪиұЎж–№жі•пјҢеҗҰеҲҷе®ғд№ҹеҝ…йЎ»еЈ°жҳҺдёәжҠҪиұЎзұ»гҖӮ
  • еұһжҖ§пјҡTrait д№ҹеҸҜд»ҘеҢ…еҗ«еұһжҖ§гҖӮеҰӮжһңеӨҡдёӘ Trait жҲ– Trait дёҺзұ»жңүеҗҢеҗҚеұһжҖ§пјҢдјҡеҜјиҮҙиҮҙе‘Ҫй”ҷиҜҜгҖӮTrait еұһжҖ§зҡ„дјҳе…Ҳзә§дёҺж–№жі•зӣёеҗҢгҖӮ
  • жһ„йҖ еҮҪж•°/жһҗжһ„еҮҪж•°пјҡTrait дёҚиғҪе®ҡд№үжһ„йҖ еҮҪж•°жҲ–жһҗжһ„еҮҪж•°гҖӮеҰӮжһң Trait дёӯеҢ…еҗ« __construct ж–№жі•пјҢе®ғдёҚдјҡиў« PHP иҜҶеҲ«дёәзү№ж®Ҡзҡ„жһ„йҖ еҮҪж•°пјҢиҖҢеҸӘжҳҜдёҖдёӘжҷ®йҖҡж–№жі•гҖӮзұ»зҡ„жһ„йҖ еҮҪж•°д»Қз„¶жҳҜ __constructгҖӮ
  • static еұһжҖ§е’Ңж–№жі•пјҡTrait еҸҜд»ҘеҢ…еҗ« static еұһжҖ§е’Ңж–№жі•гҖӮеҪ“е®ғд»¬иў«ж··е…Ҙзұ»дёӯж—¶пјҢе°ұеғҸзұ»иҮӘиә«е®ҡд№үзҡ„дёҖж ·пјҢеҸҜд»Ҙиў« self:: жҲ– static:: и®ҝй—®гҖӮ

PHP зҡ„ Trait жҳҜдёҖдёӘејәеӨ§зҡ„д»Јз ҒеӨҚз”Ёе·Ҙе…·пјҢе®ғйҖҡиҝҮвҖңеӨҚеҲ¶зІҳиҙҙвҖқжңәеҲ¶е°Ҷж–№жі•ж··е…Ҙзұ»дёӯгҖӮзҗҶи§Је…¶ж–№жі•дјҳе…Ҳзә§и§„еҲҷ (еҪ“еүҚзұ»ж–№жі• > Trait ж–№жі• > зҲ¶зұ»ж–№жі•) жҳҜжҺҢжҸЎ Trait зҡ„еҹәзЎҖгҖӮ
