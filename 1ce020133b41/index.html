Java жіӣеһӢ (Generics) иҜҰи§Ј

еҸ‘иЎЁдәҺ2024-05-23|JavaзЁӢеәҸи®ҫи®Ў
|жҖ»еӯ—ж•°:3.1k|йҳ…иҜ»ж—¶й•ҝ:12еҲҶй’ҹ

Java жіӣеһӢ (Generics) жҳҜеңЁ JDK 5.0 дёӯеј•е…Ҙзҡ„дёҖйЎ№йҮҚиҰҒиҜӯиЁҖзү№жҖ§пјҢе®ғе…Ғи®ёеңЁе®ҡд№үзұ»гҖҒжҺҘеҸЈе’Ңж–№жі•ж—¶пјҢдҪҝз”Ёзұ»еһӢеҸӮж•° (Type Parameters) жқҘиЎЁзӨәдёҚзЎ®е®ҡзҡ„зұ»еһӢгҖӮиҝҷж ·пјҢзј–иҜ‘еҷЁеҸҜд»ҘеңЁзј–иҜ‘ж—¶еҜ№зұ»еһӢиҝӣиЎҢжЈҖжҹҘпјҢд»ҺиҖҢеңЁиҝҗиЎҢж—¶йҒҝе…Қ ClassCastException зӯүзұ»еһӢиҪ¬жҚўејӮеёёпјҢжҸҗй«ҳдәҶд»Јз Ғзҡ„зұ»еһӢе®үе…ЁжҖ§ (Type Safety)гҖҒйҮҚз”ЁжҖ§ (Reusability) е’ҢеҸҜиҜ»жҖ§ (Readability)гҖӮ

ж ёеҝғжҖқжғіпјҡJava жіӣеһӢйҖҡиҝҮеј•е…Ҙзұ»еһӢеҸӮж•°пјҢдҪҝеҫ—д»Јз ҒеҸҜд»Ҙж“ҚдҪңеҗ„з§Қзұ»еһӢзҡ„ж•°жҚ®иҖҢж— йңҖиҝҗиЎҢж—¶ејәеҲ¶зұ»еһӢиҪ¬жҚўпјҢд»ҺиҖҢеңЁзј–иҜ‘ж—¶жҸҗдҫӣдәҶжӣҙејәзҡ„зұ»еһӢжЈҖжҹҘпјҢеҮҸе°‘дәҶиҝҗиЎҢж—¶й”ҷиҜҜпјҢе№¶жҸҗеҚҮдәҶд»Јз Ғзҡ„йҖҡз”ЁжҖ§е’Ңе®үе…ЁжҖ§гҖӮ

дёҖгҖҒдёәд»Җд№ҲйңҖиҰҒжіӣеһӢпјҹ

еңЁжіӣеһӢеҮәзҺ°д№ӢеүҚпјҢJava йӣҶеҗҲжЎҶжһ¶пјҲеҰӮ ArrayList, HashMapпјүеҸҜд»Ҙеӯҳж”ҫд»»дҪ•зұ»еһӢзҡ„еҜ№иұЎпјҢеӣ дёәе®ғд»¬ж“ҚдҪңзҡ„жҳҜ Object зұ»еһӢгҖӮиҝҷеёҰжқҘдәҶдёӨдёӘдё»иҰҒй—®йўҳпјҡ

  1. зұ»еһӢдёҚе®үе…Ёпјҡзј–иҜ‘еҷЁж— жі•жЈҖжҹҘйӣҶеҗҲдёӯеӯҳеӮЁзҡ„е®һйҷ…зұ»еһӢгҖӮеҰӮжһңд»ҺйӣҶеҗҲдёӯеҸ–еҮәдёҖдёӘеҜ№иұЎе№¶ејәеҲ¶иҪ¬жҚўдёәдёҚжӯЈзЎ®зҡ„зұ»еһӢпјҢе°ұдјҡеңЁиҝҗиЎҢж—¶жҠӣеҮә ClassCastExceptionгҖӮ
  2. д»Јз ҒеҶ—дҪҷпјҡжҜҸж¬Ўд»ҺйӣҶеҗҲдёӯеҸ–еҮәеҜ№иұЎж—¶пјҢйғҪйңҖиҰҒиҝӣиЎҢејәеҲ¶зұ»еһӢиҪ¬жҚўпјҢд»Јз Ғжҳҫеҫ—еҶ—й•ҝдё”жҳ“й”ҷгҖӮ

зӨәдҫӢпјҡжІЎжңүжіӣеһӢзҡ„й—®йўҳ

жіӣеһӢйҖҡиҝҮеңЁзј–иҜ‘ж—¶жҚ•иҺ·иҝҷдәӣзұ»еһӢй”ҷиҜҜпјҢи§ЈеҶідәҶдёҠиҝ°й—®йўҳгҖӮ

дәҢгҖҒжіӣеһӢзҡ„еҹәжң¬иҜӯжі•

жіӣеһӢзҡ„ж ёеҝғжҖқжғіжҳҜзұ»еһӢеҸӮж•°еҢ–пјҢеҚіеңЁе®ҡд№үзұ»гҖҒжҺҘеҸЈжҲ–ж–№жі•ж—¶пјҢдҪҝз”ЁдёҖдёӘжҲ–еӨҡдёӘзұ»еһӢеҸӮж•°жқҘд»Јжӣҝе®һйҷ…зҡ„зұ»еһӢгҖӮзұ»еһӢеҸӮж•°йҖҡеёёз”ЁеҚ•дёӘеӨ§еҶҷеӯ—жҜҚиЎЁзӨәпјҢеҰӮ E (Element), T (Type), K (Key), V (Value), N (Number) зӯүгҖӮ

2.1 жіӣеһӢзұ» (Generic Class)

е®ҡд№үдёҖдёӘзұ»ж—¶пјҢеҸҜд»ҘеңЁзұ»еҗҚеҗҺдҪҝз”Ё <T> жқҘеЈ°жҳҺдёҖдёӘжҲ–еӨҡдёӘзұ»еһӢеҸӮж•°гҖӮ

2.2 жіӣеһӢжҺҘеҸЈ (Generic Interface)

жҺҘеҸЈд№ҹеҸҜд»ҘеЈ°жҳҺзұ»еһӢеҸӮж•°пјҢе…¶е®һзҺ°зұ»еңЁе®һзҺ°жіӣеһӢжҺҘеҸЈж—¶пјҢеҸҜд»ҘжҢҮе®ҡе…·дҪ“зҡ„зұ»еһӢпјҢд№ҹеҸҜд»Ҙз»§з»ӯдҝқжҢҒжіӣеһӢгҖӮ

2.3 жіӣеһӢж–№жі• (Generic Method)

еңЁж–№жі•зӯҫеҗҚдёӯеЈ°жҳҺзұ»еһӢеҸӮж•°пјҢдҪҝе…¶еҸҜд»ҘзӢ¬з«ӢдәҺзұ»жң¬иә«зҡ„жіӣеһӢзұ»еһӢгҖӮ

  • зұ»еһӢеҸӮж•° T ж”ҫеңЁж–№жі•иҝ”еӣһзұ»еһӢд№ӢеүҚгҖӮ
  • йқҷжҖҒж–№жі•д№ҹеҸҜд»ҘжҳҜжіӣеһӢж–№жі•пјҢеӣ дёәе®ғд»¬дёҚдҫқиө–дәҺзұ»зҡ„е®һдҫӢгҖӮ

дёүгҖҒзұ»еһӢйҖҡй…Қз¬Ұ (Wildcards)

зұ»еһӢйҖҡй…Қз¬Ұ (?) з”ЁдәҺеӨ„зҗҶжіӣеһӢзұ»еһӢд№Ӣй—ҙзҡ„е…је®№жҖ§й—®йўҳпјҢзү№еҲ«жҳҜеҪ“жіӣеһӢзұ»дҪңдёәж–№жі•еҸӮж•°ж—¶гҖӮе®ғиЎЁзӨәжңӘзҹҘзұ»еһӢгҖӮ

3.1 ж— з•ҢйҖҡй…Қз¬Ұ (<?>)

иЎЁзӨәд»»ж„Ҹзұ»еһӢпјҢзӯүеҗҢдәҺ <Object>гҖӮйҖҡеёёз”ЁдәҺпјҡ

  1. дҪ дёҚзҹҘйҒ“жҲ–дёҚе…іеҝғйӣҶеҗҲдёӯеӯҳеӮЁзҡ„зұ»еһӢгҖӮ
  2. еҜ№йӣҶеҗҲиҝӣиЎҢеҸӘиҜ»ж“ҚдҪңж—¶ (еӣ дёәдҪ дёҚзҹҘйҒ“е…·дҪ“зұ»еһӢпјҢжүҖд»ҘдёҚиғҪе®үе…Ёең°ж·»еҠ е…ғзҙ )гҖӮ

3.2 дёҠз•ҢйҖҡй…Қз¬Ұ (<? extends T>)

иЎЁзӨәзұ»еһӢеҝ…йЎ»жҳҜ T жҲ– T зҡ„еӯҗзұ»гҖӮ

  • иҜ»еҸ–ж•°жҚ® (Producer Extends)пјҡдҪ еҸҜд»Ҙд»Һ List<? extends T> дёӯиҜ»еҸ–зұ»еһӢдёә T жҲ–е…¶еӯҗзұ»зҡ„е…ғзҙ гҖӮ
  • дёҚиғҪж·»еҠ ж•°жҚ®пјҡеӣ дёәзј–иҜ‘еҷЁдёҚзҹҘйҒ“ ? зҡ„е…·дҪ“еӯҗзұ»еһӢпјҢдёәдәҶдҝқиҜҒзұ»еһӢе®үе…ЁпјҢйҷӨдәҶ null д№ӢеӨ–пјҢдёҚиғҪеҗ‘е…¶дёӯж·»еҠ д»»дҪ•е…ғзҙ гҖӮ

3.3 дёӢз•ҢйҖҡй…Қз¬Ұ (<? super T>)

иЎЁзӨәзұ»еһӢеҝ…йЎ»жҳҜ T жҲ– T зҡ„зҲ¶зұ»гҖӮ

  • ж·»еҠ ж•°жҚ® (Consumer Super)пјҡдҪ еҸҜд»Ҙеҗ‘ List<? super T> дёӯж·»еҠ зұ»еһӢдёә T жҲ– T зҡ„еӯҗзұ»зҡ„е…ғзҙ гҖӮ
  • иҜ»еҸ–ж•°жҚ®пјҡд»ҺеҲ—иЎЁдёӯиҜ»еҸ–еҮәзҡ„е…ғзҙ еҸӘиғҪиў«еҪ“дҪң Object еӨ„зҗҶ (еӣ дёәдҪ еҸӘзҹҘйҒ“е®ғжҳҜ T зҡ„зҲ¶зұ»пјҢе…·дҪ“жҳҜе“ӘдёӘзҲ¶зұ»дёҚзЎ®е®ҡ)гҖӮ

3.4 PECS еҺҹеҲҷ (Producer-Extends, Consumer-Super)

иҝҷжҳҜдёҖдёӘз”ЁдәҺжҢҮеҜјдҪ•ж—¶дҪҝз”Ё extends е’Ң super йҖҡй…Қз¬Ұзҡ„еҠ©и®°з¬Ұпјҡ

  • еҰӮжһңдҪ йңҖиҰҒд»ҺжіӣеһӢз»“жһ„дёӯиҜ»еҸ– (Produce) ж•°жҚ®пјҢдҪҝз”Ё <? extends T>гҖӮ
  • еҰӮжһңдҪ йңҖиҰҒеҗ‘жіӣеһӢз»“жһ„дёӯеҶҷе…Ҙ (Consume) ж•°жҚ®пјҢдҪҝз”Ё <? super T>гҖӮ

еӣӣгҖҒжіӣеһӢж“ҰйҷӨ (Type Erasure)

Java жіӣеһӢжҳҜзј–иҜ‘ж—¶зұ»еһӢжЈҖжҹҘзҡ„пјҢиҝҷж„Ҹе‘ізқҖеңЁиҝҗиЎҢж—¶пјҢжүҖжңүзҡ„жіӣеһӢдҝЎжҒҜйғҪдјҡиў«ж“ҰйҷӨ (Erasure)гҖӮJVM зңӢеҲ°зҡ„еҸӘжҳҜеҺҹе§Ӣзұ»еһӢ (Raw Type)пјҢйҖҡеёёжҳҜзұ»еһӢеҸӮж•°зҡ„з¬¬дёҖдёӘдёҠз•ҢпјҲеҰӮжһңжІЎжңүжҢҮе®ҡдёҠз•ҢпјҢеҲҷдёә ObjectпјүгҖӮ

иҝҷж„Ҹе‘ізқҖпјҡ

  1. ж— жі•еңЁиҝҗиЎҢж—¶иҺ·еҸ–жіӣеһӢзұ»еһӢдҝЎжҒҜпјҡдҫӢеҰӮпјҢдҪ дёҚиғҪеңЁиҝҗиЎҢж—¶еҲӨж–ӯ List<String> е’Ң List<Integer> жҳҜдёҚеҗҢзҡ„зұ»еһӢгҖӮе®ғд»¬йғҪиў«ж“ҰйҷӨдёә ListгҖӮ
  2. дёҚиғҪеҲӣе»әжіӣеһӢзұ»еһӢзҡ„е®һдҫӢпјҡnew T() жҳҜдёҚе…Ғи®ёзҡ„гҖӮ
  3. дёҚиғҪеҲӣе»әжіӣеһӢж•°з»„пјҡnew T[size] жҳҜдёҚе…Ғи®ёзҡ„гҖӮ
  4. instanceof иҝҗз®—з¬ҰдёҚиғҪз”ЁдәҺжіӣеһӢзұ»еһӢпјҡobj instanceof List<String> жҳҜзј–иҜ‘й”ҷиҜҜзҡ„гҖӮ
  5. жіӣеһӢж–№жі•йҮҚиҪҪпјҡз”ұдәҺж“ҰйҷӨпјҢvoid method(List<String> list) е’Ң void method(List<Integer> list) еңЁзј–иҜ‘еҗҺзӯҫеҗҚзӣёеҗҢпјҢеҜјиҮҙж— жі•йҮҚиҪҪгҖӮ

жіӣеһӢж“ҰйҷӨзҡ„е®һзҺ°еҺҹзҗҶпјҡ

  • зұ»еһӢеҸӮж•°жӣҝжҚўпјҡжүҖжңүзұ»еһӢеҸӮж•°йғҪиў«жӣҝжҚўдёәе…¶з¬¬дёҖдёӘдёҠз•ҢпјҲеҰӮжһңжІЎжңүжҢҮе®ҡпјҢеҲҷдёә ObjectпјүгҖӮ
  • жҸ’е…ҘејәеҲ¶зұ»еһӢиҪ¬жҚўпјҡеңЁд»ҺжіӣеһӢз»“жһ„дёӯеҸ–еҮәе…ғзҙ зҡ„ең°ж–№пјҢзј–иҜ‘еҷЁдјҡиҮӘеҠЁжҸ’е…Ҙеҝ…иҰҒзҡ„ејәеҲ¶зұ»еһӢиҪ¬жҚўпјҢд»ҘзЎ®дҝқзұ»еһӢе®үе…ЁгҖӮ

дә”гҖҒжіӣеһӢзҡ„дёҖдәӣйҷҗеҲ¶

йҷӨдәҶзұ»еһӢж“ҰйҷӨеёҰжқҘзҡ„йҷҗеҲ¶еӨ–пјҢжіӣеһӢиҝҳжңүе…¶д»–дёҖдәӣдҪҝз”ЁйҷҗеҲ¶пјҡ

  1. дёҚиғҪз”Ёеҹәжң¬ж•°жҚ®зұ»еһӢе®һдҫӢеҢ–жіӣеһӢпјҡList<int> жҳҜдёҚе…Ғи®ёзҡ„пјҢеҝ…йЎ»дҪҝз”ЁеҢ…иЈ…зұ»пјҢеҰӮ List<Integer>гҖӮ
  2. дёҚиғҪжҚ•иҺ·жіӣеһӢзұ»зҡ„е®һдҫӢпјҡcatch (MyException<T> e) жҳҜдёҚе…Ғи®ёзҡ„гҖӮ
  3. дёҚиғҪеңЁйқҷжҖҒеӯ—ж®өдёҠдҪҝз”ЁжіӣеһӢзұ»еһӢеҸӮж•°пјҡstatic T instance; жҳҜдёҚе…Ғи®ёзҡ„пјҢеӣ дёәйқҷжҖҒжҲҗе‘ҳеұһдәҺзұ»пјҢдёҺд»»дҪ•зү№е®ҡзҡ„жіӣеһӢе®һдҫӢж— е…ігҖӮ
  4. жіӣеһӢзұ»еһӢдёҚиғҪжҳҜејӮеёёзұ»пјҡclass MyException<T> extends Exception жҳҜдёҚе…Ғи®ёзҡ„гҖӮ

е…ӯгҖҒжҖ»з»“

Java жіӣеһӢжҳҜзҺ°д»Ј Java зј–зЁӢдёӯдёҚеҸҜжҲ–зјәзҡ„дёҖйғЁеҲҶпјҢе®ғйҖҡиҝҮеңЁзј–иҜ‘ж—¶еј•е…Ҙзұ»еһӢжЈҖжҹҘпјҢжһҒеӨ§ең°жҸҗеҚҮдәҶд»Јз Ғзҡ„зұ»еһӢе®үе…ЁжҖ§гҖҒеҸҜиҜ»жҖ§е’ҢйҮҚз”ЁжҖ§гҖӮиҷҪз„¶е…¶еә•еұӮзҡ„зұ»еһӢж“ҰйҷӨжңәеҲ¶еёҰжқҘдәҶдёҖдәӣйҷҗеҲ¶пјҢдҪҶйҖҡиҝҮеҗҲзҗҶеҲ©з”Ёзұ»еһӢеҸӮж•°гҖҒйҖҡй…Қз¬ҰпјҲзү№еҲ«жҳҜйҒөеҫӘ PECS еҺҹеҲҷпјүпјҢд»ҘеҸҠзҗҶи§Је…¶е·ҘдҪңеҺҹзҗҶпјҢејҖеҸ‘иҖ…еҸҜд»Ҙзј–еҶҷеҮәжӣҙеҒҘеЈ®гҖҒжӣҙзҒөжҙ»зҡ„жіӣеһӢд»Јз ҒгҖӮжҺҢжҸЎжіӣеһӢжҳҜзј–еҶҷй«ҳиҙЁйҮҸ Java д»Јз Ғзҡ„е…ій”®жҠҖиғҪд№ӢдёҖгҖӮ
