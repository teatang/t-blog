> 在音乐理论和声学中，**乐音 (Musical Tone)** 与 **噪音 (Noise)** 是构成声音世界的两大基本要素。它们的根本区别在于声音振动的**规律性**和由此产生的**音高明确性**。**乐音**指具有明确音高、由规律性、周期性振动产生的复合音；而**噪音**则指没有明确音高、由不规律性、非周期性振动产生的声音。理解两者的本质差异及其在音乐中的功能，对于作曲、演奏、声学分析乃至音乐欣赏都至关重要。

{% note info %}
核心思想：
-   **本质区别**：振动是否具有周期性、规律性。
-   **音高**：乐音具有明确音高，噪音无明确音高。
-   **频率频谱**：乐音有清晰的泛音列，噪音频率分布宽泛、无规律。
-   **音乐功能**：乐音是旋律与和声的基石；噪音是节奏、织体和特殊音效的重要组成部分。
-   **动态边界**：两者并非绝对对立，在现代音乐中存在大量交融。
{% endnote %}

## 一、乐音 (Musical Tone)

### 1.1 定义

**乐音**是指那些具有**明确的音高**、**清晰的音色**，且其声波振动具有**周期性**和**规律性**的声音。这些声音通常可以被准确地记录在五线谱上，并用于构建旋律、和声和节奏。

### 1.2 物理特性与声学原理

1.  **规律性振动 (Periodic Vibration)**：
    *   乐音的声波波形是周期性的，这意味着波形会以固定的时间间隔重复。
    *   例如，弦乐器的弦、管乐器的气柱、人声的声带，都以相对稳定的频率振动。
2.  **明确的音高 (Definite Pitch)**：
    *   由于规律性振动，乐音具有一个可感知的**基频 (Fundamental Frequency)**。这个基频决定了我们听到的音高。
    *   例如，A4 音的基频是 440 Hz。
3.  **泛音列 (Harmonic Series)**：
    *   除了基频外，乐音还伴随着一系列频率是基频整数倍的**泛音 (Overtones) 或谐波 (Harmonics)**。
    *   这些泛音共同构成了乐音的**音色 (Timbre)**。不同的乐器产生不同强度的泛音，因此即使演奏相同的音高，听起来也会有不同的音色。
    *   **数学表示**：如果基频为 $f_1$，则其泛音频率为 $f_n = n \times f_1$ ($n = 2, 3, 4, \dots$)。
4.  **稳定音色 (Stable Timbre)**：
    *   乐音的泛音结构相对稳定，使其音色在持续过程中保持一致。

### 1.3 乐音在音乐中的作用

乐音是西方音乐体系的基石，承担着核心的构建作用：

*   **旋律 (Melody)**：由一系列有组织的、具有明确音高的乐音构成。
*   **和声 (Harmony)**：两个或更多乐音同时发声形成的和弦。乐音的泛音列是和声学中协和与不协和关系的重要理论基础。
*   **节奏 (Rhythm)**：虽然乐音本身不直接定义节奏，但乐音的时值和排列构成了节奏模式。
*   **调性 (Tonality)**：乐音的音高关系定义了音阶、调式和调性中心。

### 1.4 乐音的示例

*   钢琴、吉他、小提琴等弦乐器的声音。
*   长笛、单簧管、小号、圆号等管乐器的声音。
*   人声（歌唱或有明确音高的说话声）。
*   木琴、马林巴、定音鼓等具有明确音高的打击乐器。

## 二、噪音 (Noise)

### 2.1 定义

**噪音**是指那些**没有明确音高**、**模糊不清的音色**，且其声波振动具有**不规律性**和**非周期性**的声音。它们通常不能被准确地记录在五线谱上，但在音乐中仍扮演着重要的角色。

### 2.2 物理特性与声学原理

1.  **不规律性振动 (Aperiodic Vibration)**：
    *   噪音的声波波形是不规律的，不以固定的时间间隔重复。
    *   例如，敲击鼓面、沙锤摇动、风声或摩擦声。
2.  **不明确的音高 (Indefinite Pitch)**：
    *   由于缺乏规律性振动，噪音没有可感知的基频，或者其频率分布过于宽泛和随机，导致无法辨识出特定的音高。
    *   听众无法判断噪音是“高”还是“低”的特定音。
3.  **宽广的频率频谱 (Broad Frequency Spectrum)**：
    *   噪音的能量分布在非常广泛的频率范围内，没有清晰的泛音列结构。
    *   例如，白噪音 (White Noise) 在整个可听频率范围内具有平均的能量分布。
4.  **瞬态或不稳定音色 (Transient or Unstable Timbre)**：
    *   许多噪音是瞬态的（如敲击声），音色变化迅速。
    *   即使是持续性的噪音（如嘶嘶声），其音色也可能是不稳定的，缺乏乐音那种有组织的泛音结构。

### 2.3 噪音在音乐中的作用

噪音在音乐中主要用于：

*   **节奏 (Rhythm)**：许多无音高打击乐器（如小军鼓、大鼓、钹）主要通过其瞬态的音响效果来强调节奏和律动。
*   **织体 (Texture)**：噪音可以为音乐添加独特的音响色彩和质感，丰富音乐的听觉体验。
*   **氛围 (Atmosphere)**：环境噪音、电子噪音或特殊效果音可以营造特定的情绪和场景。
*   **特殊音效 (Special Effects)**：电影配乐、电子音乐和实验音乐中经常利用噪音来创造非传统的音响效果。

### 2.4 噪音的示例

*   大鼓、小军鼓、钹、沙锤、三角铁等无音高打击乐器的声音。
*   刮擦声、嘶嘶声、轰鸣声、砰击声。
*   环境音效（如风声、雨声、交通声）。
*   电子合成器中的白噪音、粉红噪音。

## 三、乐音与噪音的本质区别与比较

下表总结了乐音和噪音之间的主要区别：

| 特性         | 乐音 (Musical Tone)                       | 噪音 (Noise)                                |
| :----------- | :---------------------------------------- | :------------------------------------------ |
| **振动性质** | 周期性、规律性振动                        | 非周期性、不规律性振动                      |
| **音高**     | 明确的音高（可感知基频）                  | 无明确音高（无稳定基频）                    |
| **频率频谱** | 由基频和清晰的泛音列组成                  | 频率分布宽泛、随机、无规律                  |
| **音色**     | 稳定、清晰、和谐（由泛音结构决定）        | 不稳定、模糊、混乱或瞬态                    |
| **波形**     | 周期性、可预测的波形（如正弦波、复合波）  | 不规则、随机、不可预测的波形                |
| **音乐功能** | 构成旋律、和声、调性、协和音响的主体      | 强调节奏、增强织体、创造特殊音效、氛围      |
| **听觉感受** | 悦耳、和谐、有方向感                      | 可能刺耳、杂乱、或提供背景衬托              |

## 四、乐音与噪音的界限与交叉

乐音和噪音的区分并非绝对的二元对立，它们之间存在一个连续的频谱，并且在音乐实践中常常相互融合：

1.  **“灰区”乐器**：
    *   **定音鼓 (Timpani)**：虽然是打击乐器，但它能调音并发出明确的音高，介于乐音和噪音之间。
    *   **手碟 (Handpan)**：能发出丰富泛音的音高，同时也有打击的噪音成分。
    *   **锣 (Gong)** 和 **钹 (Cymbal)**：大部分是无音高的噪音，但高质量的锣和钹也可能含有复杂的谐波，在特定情境下能营造出类似有音高的效果。
2.  **现代音乐的拓展**：
    *   **20世纪和21世纪音乐**：作曲家们常常有意模糊乐音与噪音的界限。例如，通过**延伸演奏技巧 (Extended Techniques)**，使乐器发出非传统音色，如弦乐器拉奏弓杆（噪音）、管乐器吹奏气声（噪音）。
    *   **电子音乐和具体音乐 (Musique Concrète)**：直接使用录制的自然噪音、环境音效作为音乐素材，与乐音进行拼接和组合。
    *   **音块 (Tone Clusters)**：密集且高度不协和的乐音组合，在听觉上接近于噪音。
3.  **噪音的审美化**：
    *   在工业音乐、噪音音乐等流派中，噪音本身被提升为音乐审美的主体，不再是乐音的陪衬。
    *   许多音高不明确的打击乐器，其独特音色和动态效果被视为音乐表现力的重要组成部分。

## 五、在音乐实践中的应用

### 5.1 作曲与配器

*   作曲家通过精确的**音高、节奏、和声**（主要基于乐音）来构建音乐结构。
*   同时，他们利用**噪音**（如打击乐、特殊音效）来增强节奏感、提供音色对比、营造氛围或引入戏剧性元素。例如，交响乐中的钹声通常用于高潮的强调，而不是作为旋律的一部分。

### 5.2 演奏与技术

*   演奏乐音要求精确的**音准、音色控制和乐句处理**。
*   演奏噪音则需要掌握特定的**打击技巧、摩擦方式或呼吸控制**，以产生所需的音响效果。

### 5.3 声学与音频工程

*   在录音和混音中，音频工程师需要区分乐音和噪音。
*   **降噪技术**主要针对不需要的噪音，以提升乐音的清晰度。
*   **音效设计**则专注于合成和处理各种噪音，以创造特定的听觉体验。

## 六、总结

乐音与噪音是构成我们听觉世界和音乐艺术的两个基本维度。乐音以其规律性、明确的音高和丰富的泛音列，承载了旋律、和声和调性的核心功能；而噪音则以其非规律性、无明确音高和独特的音色，为音乐注入了节奏、色彩、氛围和原始的冲击力。

理解这两种声音的本质和相互关系，能够帮助我们更深入地分析和欣赏音乐作品，无论是传统经典还是先锋实验。在当代音乐中，乐音与噪音的界限日益模糊，艺术家们不断探索它们交融的可能性，拓展着音乐表现的疆界，使声音的调色板变得更加广阔和丰富。
