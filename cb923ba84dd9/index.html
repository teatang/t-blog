Python NumPyиҜҰи§Јпјҡз§‘еӯҰи®Ўз®—зҡ„еҹәзҹі

еҸ‘иЎЁдәҺ2023-02-15|Pythonеә“
|жҖ»еӯ—ж•°:2.9k|йҳ…иҜ»ж—¶й•ҝ:12еҲҶй’ҹ

NumPy (Numerical Python) жҳҜ Python дёӯз”ЁдәҺз§‘еӯҰи®Ўз®—зҡ„ж ёеҝғеә“гҖӮе®ғжҸҗдҫӣдәҶдёҖдёӘй«ҳжҖ§иғҪзҡ„еӨҡз»ҙж•°з»„еҜ№иұЎ ndarrayпјҢд»ҘеҸҠз”ЁдәҺеӨ„зҗҶиҝҷдәӣж•°з»„зҡ„е·Ҙе…·гҖӮNumPy жҳҜ Python ж•°жҚ®з§‘еӯҰе’ҢжңәеҷЁеӯҰд№ з”ҹжҖҒзі»з»ҹзҡ„еҹәзҹіпјҢи®ёеӨҡе…¶д»–еә“пјҲеҰӮ SciPy, Pandas, Matplotlib, Scikit-learnпјүйғҪе»әз«ӢеңЁ NumPy ж•°з»„д№ӢдёҠгҖӮ

ж ёеҝғжҖқжғіпјҡNumPy еј•е…ҘдәҶй«ҳж•Ҳзҡ„ ndarray ж•°жҚ®з»“жһ„пјҢйҖҡиҝҮеҗ‘йҮҸеҢ–ж“ҚдҪңжҳҫи‘—жҸҗеҚҮдәҶ Python еӨ„зҗҶж•°еҖјж•°жҚ®зҡ„жҖ§иғҪгҖӮ

дёҖгҖҒдёәд»Җд№ҲйҖүжӢ© NumPyпјҹ

Python иҜӯиЁҖжң¬иә«еӨ„зҗҶеҲ—иЎЁзӯүж•°жҚ®з»“жһ„ж—¶ж•ҲзҺҮиҫғй«ҳпјҢдҪҶеҜ№дәҺеӨ§и§„жЁЎж•°еҖји®Ўз®—иҖҢиЁҖпјҢеҺҹз”ҹзҡ„ Python еҲ—иЎЁж•ҲзҺҮдҪҺдёӢгҖӮNumPy йҖҡиҝҮд»ҘдёӢж–№ејҸи§ЈеҶідәҶиҝҷдёӘй—®йўҳпјҡ

  1. й«ҳжҖ§иғҪ ndarray еҜ№иұЎпјҡndarray еӯҳеӮЁеҗҢзұ»еһӢж•°жҚ®пјҢеңЁеҶ…еӯҳдёӯиҝһз»ӯеӯҳеӮЁпјҢзӣёжҜ” Python еҲ—иЎЁпјҢеҚ з”Ёзҡ„еҶ…еӯҳжӣҙе°‘пјҢи®ҝй—®йҖҹеәҰжӣҙеҝ«гҖӮ
  2. еҗ‘йҮҸеҢ–ж“ҚдҪңпјҡNumPy е…Ғи®ёеҜ№ж•ҙдёӘж•°з»„иҝӣиЎҢж“ҚдҪңпјҢиҖҢж— йңҖзј–еҶҷжҳҫејҸзҡ„еҫӘзҺҜгҖӮиҝҷдәӣж“ҚдҪңйҖҡеёёеңЁ C жҲ– Fortran дёӯе®һзҺ°пјҢжү§иЎҢйҖҹеәҰиҝңи¶… Python еҫӘзҺҜгҖӮ
  3. е№ҝж’ӯ (Broadcasting)пјҡNumPy иғҪеӨҹеҜ№дёҚеҗҢеҪўзҠ¶зҡ„ж•°з»„жү§иЎҢз®—жңҜиҝҗз®—пјҢжһҒеӨ§ең°з®ҖеҢ–дәҶд»Јз ҒгҖӮ
  4. дё°еҜҢзҡ„ж•°еӯҰеҮҪж•°пјҡжҸҗдҫӣдәҶеӨ§йҮҸзҡ„ж•°еӯҰеҮҪж•°пјҢеҸҜд»ҘзӣҙжҺҘеә”з”ЁдәҺж•°з»„гҖӮ
  5. дёҺ C/C++/Fortran д»Јз ҒйӣҶжҲҗпјҡеҸҜд»Ҙж–№дҫҝең°йӣҶжҲҗз”Ёе…¶д»–й«ҳж•ҲиҜӯиЁҖзј–еҶҷзҡ„д»Јз ҒгҖӮ

е…«гҖҒжҖ»з»“дёҺиҝӣйҳ¶

NumPy жҳҜд»»дҪ• Python з§‘еӯҰи®Ўз®—д»»еҠЎзҡ„ж ёеҝғе·Ҙе…·гҖӮзҶҹз»ғжҺҢжҸЎ ndarray зҡ„еҲӣе»әгҖҒзҙўеј•гҖҒж“ҚдҪңе’Ңеҗ„з§ҚеҠҹиғҪпјҢжҳҜиҝӣиЎҢж•°жҚ®еҲҶжһҗгҖҒжңәеҷЁеӯҰд№ е’Ңз§‘еӯҰз ”з©¶зҡ„еҹәзЎҖгҖӮ

иҝӣйҳ¶ж–№еҗ‘пјҡ

  • жӣҙеӨҡж•°еӯҰеҮҪж•°пјҡжҺўзҙў np.sin(), np.cos(), np.exp(), np.log() зӯүйҖҡз”ЁеҮҪж•° (Universal Functions, ufuncs)гҖӮ
  • ж–Үд»¶ I/OпјҡдҪҝз”Ё np.save(), np.load(), np.savetxt(), np.loadtxt() зӯүеҮҪж•°иҜ»еҶҷж•°жҚ®гҖӮ
  • з»“жһ„еҢ–ж•°з»„пјҡеҲӣе»әеёҰжңүдёҚеҗҢж•°жҚ®зұ»еһӢеӯ—ж®өзҡ„еӨҚжқӮж•°з»„гҖӮ
  • еҶ…еӯҳи§ҶеӣҫдёҺеүҜжң¬пјҡзҗҶи§ЈдҪ•ж—¶ NumPy ж“ҚдҪңиҝ”еӣһи§ҶеӣҫпјҲе…ұдә«ж•°жҚ®пјүе’ҢеүҜжң¬пјҲзӢ¬з«Ӣж•°жҚ®пјүпјҢиҝҷеҜ№дәҺйҒҝе…Қж„ҸеӨ–дҝ®ж”№е’ҢдјҳеҢ–еҶ…еӯҳдҪҝз”ЁиҮіе…ійҮҚиҰҒгҖӮ
  • дёҺ Pandas з»“еҗҲпјҡNumPy ж•°з»„жҳҜ Pandas DataFrame е’Ң Series зҡ„еә•еұӮж•°жҚ®з»“жһ„гҖӮ
  • дёҺ Matplotlib еҸҜи§ҶеҢ–пјҡдҪҝз”Ё NumPy ж•°з»„з”ҹжҲҗж•°жҚ®е№¶з”Ё Matplotlib иҝӣиЎҢеҸҜи§ҶеҢ–гҖӮ
